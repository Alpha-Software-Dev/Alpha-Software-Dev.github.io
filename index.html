What is Fractal OS?

Fractal OS is a graphical shell for Windows 10. It can be used by people who have poor computer skills and who need only basic computer programs.

Some innovations

We putted all apps in one meta-app, what is realy cool.
Here are some reasons.
1)It is not very large (150mb)
2)It is easy to install
3)It helps to sort system files
