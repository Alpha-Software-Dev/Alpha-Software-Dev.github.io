Информация о приёме в лицей

Главное о приёме:

  • Лицей приглашает на вакантные места для обучения в профильных классах (5-10) Подробнее о профилях на странице "Обучение".
    •
  • Для зачисления необходимо иметь отметки "4" и "5" по профильным предметам, а также успешно сдать тестирование.
    •
  • Для зачисления в Художественно-Технологические классы требуется предоставить комиссии портфолио творческих работ.
    •
  • По результатам тестирования комиссией будет проведён индивидуальный отбор. В состав комиссии входят: заместители директора по УВР, руководители предметных кафедр, классные руководители, учителя - предметники.
    •
  • День открытых дверей для учащихся и их родителей обычно проводится в марте на весенних каникулах.
    •
